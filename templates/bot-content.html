LOTRO Guides Bot

Look up quests, deeds, items, guides, stat caps, and trait builds — right from Discord.

Add to Server

Slash Commands

/quest

Look up any quest by name. Shows level, zone, arc, bestower, rewards, and quest chain links.

/deed

Search deeds with type-specific icons. Shows objectives, rewards (LP, Virtue XP, Reputation), and region info.

/item

Look up items from the full database. Shows item type, stats, and icon thumbnail.

/guide

Link to any guide on the site — raids, instances, crafting, class builds, and more.

/build

Browse class trait builds — both guide builds and community-submitted builds with like counts.

/statcaps

Calculate stat caps for any class, level, and penetration preset using Giseldah formulas.

/map

Link to an interactive map region — Ered Luin, Moria, Mordor, and more.

Features

  • ✨ Autocomplete — Start typing and get instant suggestions for quests, deeds, items, and guides
  • 🎨 Custom LOTRO Emojis — Class icons, deed type icons, LOTRO Points, Virtue XP, and The One Ring
  • 🔗 Direct Links — Every embed links back to the full detail page on lotroguides.com
  • ⚡ Fast Responses — Backed by serverless functions with deferred follow-up for complex queries
  • 🛡️ Secure — Ed25519 signature verification on every interaction
